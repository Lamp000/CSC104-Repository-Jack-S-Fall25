CSC104 HTML Basics Page

HTML stands for Hypertext Markup Language. HTML provides the structure of the webpage by marking up the text. HTML does not have many rules, but does have some best practices, such as only 1 h1 tag on a page.

History of HTML

  • HTML 1.0 - developed in early 1990's
  • HTML 2.0 - released in 1995
  • HTML 3.2 - released in 1997
  • HTML 4.01 - released in 1999
  • HTML 5 released in 2014 (current version) - living standard

Cascading Style Systems (CSS)

CSS is a language that is used to make HTML look better. CSS can control the layout, colors, font - any visual element on the page changed from the default look using CSS

History of CSS

  • CSS1 released in 1996
  • CSS2 released in 1998
  • CSS3 released in 2011 - this is the current version - living standard

Basic structure

    selector{
        property: value;
    }

Selector optio0ns

  • HTML element
  • Custom class - starts with a .
  • Id of an HTML element - start with a #

Ways to use CSS

  • External CSS - this is a seperate file - best practice
  • Embedded CSS - the CSS styles are on a single page
  • Inline CSS - the styles are created on each HTML element
